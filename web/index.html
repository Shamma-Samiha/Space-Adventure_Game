Space Adventure

Steer the shuttle, dodge monsters, collect coins, and reach the sun!

Download for Windows Run from source

How to Play

  • Arrow keys to move the shuttle.
  • Avoid monsters; they wander then chase.
  • Collect coins for points and reach the bright sun to win.
  • Three lives; difficulty ramps up over time.

Run on Windows

  1. Install Python 3.10+ and Pygame: pip install pygame.
  2. Run the game: python ../game/space_adventure.py.
  3. Enter name and age, then play with the arrow keys.

Build an EXE

  1. Install PyInstaller: pip install pyinstaller.
  2. From the game folder, build: pyinstaller --noconsole --onefile --name SpaceAdventure --add-data "assets;assets" space_adventure.py
  3. Find the executable at game/dist/SpaceAdventure.exe.
